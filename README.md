🌌 Project Overview:
This project serves as a deep dive into CSS animation performance and absolute positioning. Every planet is placed in a container that rotates around the Sun, simulating orbital periods with varied animation speeds.

🚀 Key Features:
Pure CSS: Zero JavaScript. All motion is handled by the browser's GPU via transform: rotate().

Starfield Generation: Uses a massive box-shadow on a single pixel to create a randomized, high-performance background of stars.
Technical Stack:
HTML5: Semantic structure for the Sun and Planet containers.
CSS3:   @keyframes for infinite loops.
        border-radius: 50% for circular paths.
        box-shadow for planetary glow and depth.
